# AgentScope Sandbox Images

This repository contains Docker images for AgentScope sandbox environments.

## Images

- agentscope-sandbox-base.tar (944MB)
- agentscope-sandbox-browser.tar (2.2GB)
- agentscope-sandbox-filesystem.tar (2.2GB)
- agentscope-sandbox-gui.tar (2.3GB)
- agentscope-sandbox-mobile.tar (2.8GB)

## Usage

These are Docker image archives that can be loaded using:
```bash
docker load -i <image-file>.tar
```
